# DS2030 – Data Structures and Algorithms for Data Science

## Course Info

| | |
| --- | --- |
| **Course Name** | Data Structures and Algorithms for Data Science |
| **Course Id** | DS2030 |
| **Course Period** | Fall 2025 |
| **Instructor** | C.K. Narayan |
| **Description** | Data structures and algorithms play a crucial role in computing applications that involve data management. They are particularly vital for data-driven decision systems such as the Web, Databases, Machine Learning, Computer Vision, and Artificial Intelligence systems. This introductory course explores fundamental and commonly used data structures, emphasizing both theoretical analysis and practical implementation in Python. |

## Programming Focus

- Python-based implementation of data structures and algorithms  
- Emphasis on algorithmic thinking and time–space complexity analysis  
- Applications in data-centric systems, machine learning, and AI pipelines
